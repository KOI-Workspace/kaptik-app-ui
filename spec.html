Kaptik 모바일 앱 (프로토타입 기준) · K-pop 라이브/영상 실시간 다국어 자막 서비스
PC 확장 프로그램 버전과 달리, 모바일은 PIP·우측 패널 대신 앱 내부에 영상을 임베드하고 하단 바텀시트로 자막을 표시한다. 하단 내비게이션은 Home / Translate / My 3탭.

로그인 상태일 때 ✓ 구현됨

  • 로그인 전: 홈에서 라이브 영상을 누르거나 Translate 탭에서 링크를 제출하면 로그인 모달로 막힌다.
  • 로그인 수단은 두 가지 — Google 소셜 로그인 / 이메일·비밀번호 자체 로그인.
  • 로그인에 성공하면 원래 하려던 동작(영상 재생·링크 제출)으로 자동 복귀한다.
  • 로그인 후: My 탭에 계정 정보(이메일·로그인 수단)와 로그아웃이 표시된다.
  • 게이트 순서: 로그인 → 결제 2단계. 로그인만 하고 미결제면 이어서 결제 모달이 뜬다.

결제가 안 된 상태일 때 ✓ 구현됨 ◆ 정책

  • 우상단 Upgrade 버튼으로 결제 상태를 항상 노출.
    • 결제 전 → 퍼플 그라디언트 “Upgrade” 버튼 (누르면 결제 모달)
    • 결제 후 → “Pro” 표시로 전환 (현재 플랜·다음 결제일 확인 가능)
  • 결제 전 상태에서 라이브 영상 재생 또는 Translate 링크 제출을 시도하면 결제(Upgrade) 모달로 막는다.
  • 결제 정보·현재 플랜(Free / Pro)은 My 탭 → 멤버십/결제에서 확인.
→ Pro: 모든 영상에서 번역 제공.  Basic: 우리가 자막을 송출하는 라이브에 대해서만 사용 가능.
  • Basic 요금제로 라이브를 보면 자막이 정상 표시되지만, Basic 요금제로 일반 영상을 보면 자막이 뜨지 않는다. ◆ 정책
  • 현재 프로토타입은 Free / Pro 2단계로 구현 — Basic 분기는 정책 확정 후 반영 예정. … 예정

영상에 대한 자막을 새로 생성하고 있을 때 ✓ 구현됨

  • Translate 탭에서 URL 직접 입력 또는 플랫폼(Weverse · YouTube · X · Instagram) 선택으로 영상 제출.
  • 게이트(로그인·결제) 통과 후, 전체 화면 Generating 오버레이가 뜬다 — 스피너 + 단계별 진행 표시.
링크 분석→ 스크립트 추출→ STT (조건부)→ 다국어 번역
  • 스크립트 품질이 좋으면 STT를 건너뛴다 (예: YouTube — 공개 자막 트랙 추출 가능).
  • 스크립트가 없거나 품질이 낮으면 STT까지 함께 진행 (예: Weverse — 공개 자막 트랙이 없어 사실상 STT 경로).

자막이 생성된 직후 ✓ 구현됨

  • 자막 생성이 완료되면 “번역이 완료됐어요” 알림(토스트)을 띄운다.
  • 완료되는 대로 자막 재생(Player) 화면으로 전환되어, 영상의 현재 타임스탬프 구간에 맞는 자막을 바로 표시한다.
  • 앱이 백그라운드여도 번역 완료 시 푸시 알림으로 알려준다. … 예정

자막이 재생될 때 ✓ 구현됨

  • 상단에 영상(YouTube 임베드), 하단 바텀시트에 자막이 흐른다. (자동재생은 브라우저 정책상 음소거로 시작)
  • 자막은 영상의 실제 재생 시간에 동기화되어 한 줄씩 올라온다.
  • 자막 한 줄을 탭하면 → 해당 타임스탬프로 영상이 이동(seek)한다.
  • 시트를 위/아래로 드래그해 높이를 조절할 수 있고, 과거 자막을 스크롤하면 “↑ Latest(최신)” 버튼이 나타난다.
  • 설정 (언어 FAB)
    • 자막 언어 선택 (한국어 ↔ 영어 등 16개 언어)
    • 우측 패널(자막 시트) 켤지/끌지 선택 … 예정
    • 말하는 사람(화자) identification — 화자별 아바타·이름·색상으로 구분 (켜짐 디폴트)
  • 자막 UI 설정
    • 문화맥락 해설 — 자막 속 밑줄 단어를 탭하면 인라인 해설 카드가 펼쳐짐 (구현됨)
    • 자막 한 줄 / 두 줄 표시 선택 (두 줄 디폴트) … 예정
    • 폰트 사이즈 조절 (Slider) … 예정
    • Background Opacity 조절 (Slider) … 예정
  • 케이팝 영상이 아닌 경우(목소리 등록이 안 되어 있는 경우) 화자 구분 기능이 자동으로 꺼져 있어야 함. ◆ 정책
  • 음질이 너무 안 좋거나 사람이 많은 라이브의 경우에도 화자 구분 기능이 자동으로 꺼져 있음. ◆ 정책

홈 화면 & 알림 ✓ 구현됨

  • My Artists: 팔로우한 아티스트 가로 스트립, 라이브 중이면 링 강조, 끝에 + Add로 추가.
  • Live Now: 팔로우 아티스트의 라이브/영상 소식을 히어로 카드 + 리스트로 노출. 누르면 게이트를 거쳐 재생.
  • 알림 배지 + 토스트로 “○○가 라이브를 시작했어요” 알림 표현.
  • 팔로우 아티스트 라이브 시작 시 푸시 알림 발송. … 예정
